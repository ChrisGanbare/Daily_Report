# 欢迎使用 ZR Daily Report 文档

ZR Daily Report 是一个用于生成切削液设备日常库存报告的Python应用程序。

## 功能特性

- 多设备批量数据处理
- 自动化数据库查询
- Excel报表生成（含图表）
- 客户对账单自动生成
- 动态列数调整（适应不同油品数量）
- 配置文件加密存储
- 完整日志记录
